Release 1.40

We have been busy trying to produce a good product. We do need financial assistance to continue, so if you use this program, please register

Whats new

  1. Time bug fixed - another bug with areas with 30min offset from GMT. This one turned out to be a MS Visual Studio problem
  2. Sound support - choose your own sounds for time events and alarms
  3. Custom tooltips added
  4. Link to enable Online computer time Syncronization
  5. UI improvments

Features

  1. WorldTime2000 is self extracting setup program, not a ZIP file. Just click + install
  2. Windows Install feature
  3. Alarms with optional messages can be set for displayed locations
  4. Over 400 built-in locations including DST details This includes cities and time zones
  5. Font support - choose and font, point size and style to display times
  6. Wallpaper support added - choose any bitmaps for the window background. Just like the windows desktop
  7. Text will automatically resize itself to the width of the window
  8. Text and background colors user definable
  9. An animated spinning globe. It doesn't do anything, but it looks good!
  10. Right mouse button ( context ) menus implemented
  11. HTML help. That means help looks and acts just like a web page
  12. Locations can be viewed, arranged on screen, modified and deleted
  13. Display preferences are saved ( window details, colors, toolbars etc )
  14. Time is adjusted according to the Daylight Savings rules for that location
  15. 30 min and 1 hour increments from GMT
  16. Select between 12 or 24 hour time
  17. Home location's time zone set automatically. Home can be changed to any name of your liking
  18. You get to see your name on the display when you register

This evaluation version will last for 1 month. After this period, you may still run the program, but only in 'view' mode. View mode allows you to view the display, but it cannot be altered until registered.
