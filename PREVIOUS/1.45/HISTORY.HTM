WorldTime2000 History

3rd April: Version 0.50

  1. Implementation of Daylight Savings.
    - Daylight savings details for each location can be viewed. To view, press "Show Details" button in Add to locations dialog
    - If a location is in currently in Daylight Savings Time ( DST ), a small icon of a sun will appear next to that location
    - Full control of DST when adding a new location
  2. Added over 350 locations
    - All locations contain up-to-date DST information.
  3. More customizable. Spinning globe may be turned off, as well as 12 or 24 hour time.
  4. 30 min offsets from GMT has been implemented.

26th March: Version 0.21

  1. Fixed a bug where fonts would get corrupted overtime. -

26th March: Version 0.20

  1. Select between 12 or 24 hour time
  2. Home location's time zone set automatically. Home can be changed to any name of your liking
  3. Display enhanced

24th March: Version 0.10 ( Initial release )

  1. The purpose of this release is to get comments on what you think will make this a better program, as well as any bugs found.
