Release 1.0

WorldTime2000 1.0 has the following added features:

  1. WorldTime2000 is now a self extracting setup program. Just click & install
  2. Windows Install/Un-install feature
  3. Alarms can be set for displayed locations
  4. Time Zones added to list of locations
  5. Data will automatically size itself to the width of the window
  6. Text and background colors user definable
  7. A nicer spinning globe to allow easier reading
  8. Right mouse button ( context ) menus implemented
  9. Improved visual cues
  10. UI enhancements
  11. Time zone name for home now displays Standard or DST names depending if Daylight savings is currently in force.
  12. Toolbar/Status bar view state is now saved

This evaluation version will last for 1 month. After this period, you may still run the program, but only in 'view' mode. View mode allows you to view the display, but it cannot be altered until registered.

Coming in future versions

  1. Ability to choose different fonts
  2. Background pictures
  3. More powerful and customizable alarms
  4. Location list will be sortable by town, time, country etc
  5. Reside in the System Tray
