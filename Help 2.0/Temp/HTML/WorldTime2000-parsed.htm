Using WorldTime2000

Overview

Setting your Time Zone

Normally you do not need to do this as this is set when Windows is installed. If the times in WorldTime2000 look wrong, then you will need to tell your computer what time zone you reside in.

o this by choosing Date/Time in the control panel and check your time zone. Choose the Time Zone tab and make sure the selected time zone matches your home or reference time zone.

Quick start

To get you familiar with the program in no time at all, please note the following:

Ensure that your time zone is correctly set up. Do this by choosing Date/Time in the control panel and check your time zone.

To display a location, choose 'Add Location to display' button from the toolbar or Menu

Left click on a location to make it the new base reference from which hour differences are calculated A house icon will be displayed to let you know this is your new temporary 'Home'  or point of reference. Left click on Home for normal operation which displays hour differences from your home. Home is normally your place of residence and is the top most location with a boarder around it

Right click on a location to move, modify, delete, set alarms, set the font or sort the display

To exit the program, choose Exit from the menu or type X.

To send WorldTime2000 to the System Tray ( the list of icons on the bottom right of the screen ), click the Applications Close button ( the X on the upper right ) . Double click the WorldTime2000 icon in the system tray to restore the window, or right click the icon for options

Thats all there is to setting up and using WorldTime2000! If you wish to customize WorldTime2000 further, read on

Performing Actions

 

There are several ways to perform tasks in WorldTime2000. All these tasks can be achieved in 4 ways

Main menu

Context menu ( right click on a location )

Toolbar

Hotkeys

The 2 major types of tasks are action and view orientated, and is located in the Action and View menus accordingly. Actions in the main menu perform general type tasks such as saving data and setting preferences. To perform actions on an individual location on the display, right click on that location, and a context menu will pop up. The action you choose from the context menu will generally apply to that location only. The most common actions are available as buttons to select in the toolbar as well.

 

Hotkeys

The current hotkey assignments for actions are:

H - Home

D - Add Location to display

A - Add Location to database

W - What if the time was ?

P - Preferences

F - Set Font ( to set font for a specific location, right click on that location )

X - Exit Program

F1 - Help

 

The Display

Overview

The display shows a list of locations, line by line. The top most location ( surrounded by a border ) is your home. It reflects whatever you have set in Windows control panel under Date/Time as your local time zone.

Following your home is a list of locations, line by line. Each location displays the city ( and country if selected ). Following that are a series of status icons. The time at that location will appear next. This is the current time at that location. The Date appears after the time. The last item to show will be a number either, plus (+) of Minus (-). This number indicates the number of hours from your home location. The display screen has all the information you need to have in order to find the difference in time from one place to another, and the number of hours difference from one location to another. You can turn off this feature in the Preferences dialog.

If your wish to see the difference between two locations other than home, you can Left click on a location, this will underline that location, which is now your temporary 'home' or point of reference, Now all the numbers on the far right will either be plus (+) or minus (-) number of hours from your temporary 'home', including your home location.

if your mouse sits on a location for more than 2 seconds, a tool tip will pop up showing extra information for that location.

If you right click on a location you will have the option to view, delete, move, Set Font or Ser Alarms for that location.

Note: If you remove a location from the display, it does not remove that location from the locations database, just the display. To remove a location from the database, choose 'Delete' from the Locations Database dialog ( it is not recommended to do so ).

 

Status Icon Meaning

 

For each location, there can be up to 3 status icons displayed to the left of the locations time. The status icons signify

1.  Sun and night phase This gives a quick visual confirmation for the time of day or night. There are 14 different stages from dawn to sunset, twilight to dawn.

2.  Alarm Status. This icon is displayed if there are any alarms applicable for this location

3.  Daylight Savings. This icon is displayed if this location is currently in daylight savings time

Setting Home

 

      Hotkey H

      Menu Action->Set Home

      Toolbar Set Home/Reference

 

Your home is your place of residence, where you and your computer live. By default, your home is simply labeled Home, but you can change it to anything you like. For example, a town, city, state, name, company or whatever you like.  Your home appears as the top location in the display with a outline drawn around it,

Add Locations to the Display

 

      Hotkey A

      Menu Action->Add Location to display

      Toolbar Add Location to display

 

This function allows you to choose one of the many cities already programmed into WorldTime2000. Once the button has been selected a box with a list will appear. The cites are initially shown in alphabetical order by the first letter of the city. To sort the list by ascending/descending by city, country or time, click on the column headers "Location", "Country" or "GMT". These sorting preferences will be saved for all location lists, such as those in Modify Location & Delete Location, as well as Add Location to Display. Once you have highlighted a city you have the option of seeing more details about that city, by selecting the "show details" button. If this button is selected you will see a more detailed outline of the city. This will show the offset in hours from GMT and whether or not the location is subject to daylight savings. Once you have selected a location to add to the display and click Ok, this location will be added to the WorldTime2000 display screen.

 

Moving locations

 

      Hotkey None

      Menu None

      Context Menu Move

 

Right click on a location to move it. This allows you to move locations within the display. In essence, its a tool to let you order your locations on the display. When you add a location to the display, it adds the location to the bottom of the list. You may want to move that location to another position. To do so, choose Move. The location you wish to move will start flashing with a message 'Select new location'. Using the mouse, select a new slot where you would like the location to be moved to

 

What if the time was ?

 

      Hotkey W

      Menu Actions->What if the time was ?

      Context Menu What is the time was?

 

This feature allows you to play What if ? with a time of your choice. For example, suppose you had to call a friend who lived in Paris at 6pm their time. To do this, you would need to know what time to call him in your local time. Firstly, you would add Paris to your display, then right click on Paris, or Choose What if... from the actions menu. Enter the time 6pm and click Ok. All the locations in the display will now reflect their local time when it is 6pm Paris time. To let you know you are in What If ? mode, and not the current mode, a series of question marks will flash on the screen. If you click on any other the other locations, What if ? will change your new selected location to 6pm, and show all the other locations local time.

To exit this mode, or to toggle it on and off, choose Toggle What if Mode from the menu.

 

 

Alarms

 

Overview

      Hotkey A

      Menu Action->Alarms

      Toolbar Alarm Control Panel

      Context Menu->Alarms

The alarm control panel displays all the information necessary for editing alarms.

Note: You can only set an alarm for a location that is in the display.

Setting an Alarm

To set an alarm, open up the Alarm Control Panel and choose Add. Choose the location that this alarm applies to and set the necessary details. The time for the alarm to go off is in that locations local time, not your local time.

Responding to an Alarm

When an alarm goes off, a Window will pop up over the top of any application you have running showing the message details that you set when you added this alarm. If you wish to attend to this alarm later, click on Snooze. You will reminded again after several minutes, much like a regular alarm clock.

The Locations Database

Add to Locations List

      Hotkey A

      Menu None

      Toolbar Edit Locations Database->Add

This function allows you to add a location to the already large list of world cities available in WorldTime2000. You will be prompted to enter the Name/Town. EXAMPLE: Sydney or even Bob's place. Enter the country ( this is optional ). Then you will asked the Offset in hours from GMT. This is amount of time plus (+) or minus (-) of GMT. You will need to know if the city, town, etc, is currently on Daylight Savings and the start and ending dates. This can also be entered here. If you do not know if the location you wish to add is a user of daylight savings of not, you may not get a correct time reading to another location in WorldTime2000. Once you have entered your new location to add to WorldTime2000 press the ok button. You will the find you new location added to the list of locations found in the Add locations to display list. More often then not you new location will be added to the bottom of the list and not necessarily in alphabetical order.

Modifying a Location

 

      Hotkey None

      Menu None

      Toolbar Edit Locations Database->Properties

Select a location in the Edit Locations Database main dialog and click on Properties. Make the necessary modifications and press OK to save your changes.

Deleting a Location

 

      Hotkey None

      Menu None

      Toolbar Edit Locations Database->Remove

Select a location in the Edit Locations Database main dialog and click on Remove. Press Ok to confirm that you really want this location deleted. It is recommended that you not delete locations, but to remove locations that there erroneously added.

 

 

The System Tray

When WorldTime2000 is set up exactly how you want it, and you no longer need to view the main display any more, you can close or minimize the application window. At this time, WorldTime2000 will continue to run in the system tray. The system tray is the area on the bottom right of your screen containing a list of icons from different applications.

There are several things you can do with this icon.

Double left clicking on this icon will open the main WorldTime2000 window

Right clicking on this icon will present a mini-menu where you can do several things


This provides a quick way to view or set important information without opening the main window.

WorldTime2000 uses the same icon picture in the system tray as its shortcut on the desktop. If you keep the mouse over the WorldTime2000 for a few seconds, a tooltip will popup describing:

How long WorldTime2000 has been running

How many alarms are pending?

Whether your home is is currently in DST (Daylight Savings Time) or not

 

Customizing WorldTime2000

Skins

 

      Menu Action->Load Skin

      Menu Action->Remove Skin

      Toolbar

 

WorldTime2000 supports application skinning where you can change the overall look of the program by loading different skins. WorldTime2000 is compatible with standard ActiveSkin skins

To change the skin, choose Load Skin and pick a new skin. WorldTime2000 will adopt the look of the new skin immediately.

If you prefer to have the standard Windows look, choose Remove skin. Note: You will have to restart WorldTime2000 if you choose this option.

Fonts

 

      Hotkey F

      Menu Action->Fonts

      Contect Menu Set Font for this location

 

There are 2 types of Font settings. You can set the Font for all locations or set the font for an individual location. To set the font for an individual location choose Set Font for this location by right clicking on the location you wish to change.

Once you have chosen a new font, WorldTime2000 will then adjust the display to accommodate the size and height of the new font. You may have to resize the window to achieve the best affect.

Preferences Dialog Options

 

      Hotkey P

      Menu Action->Preferences

      Toolbar Preferences

 

Changing the Background

In Preferences dialog, choose the Display Tab

You can set WorldTime2000 background to be either a simple color or a wallpaper

With wallpapers, the bitmap must be a bitmap ( BMP ) file. If the bitmap file is smaller than the display, the background is drawn using a tiling method, thus making sure the background is always textured, even for small wallpapers. Alternatively, you may choose to display pictures larger than the display, to make a scenic background look. The choice is entirely up to you.

Changing Sounds

In Preferences dialog, choose the Sounds Tab

This lets you choose what sounds you wish to hear for different time events. There are currently 3 time events:

When an alarm sounds

When the time reaches the hour at your home location

When the time reaches a quarterly interval ( e.g. 4:15, 4:30, 4:45 ) at your home location

WorldTime2000 comes with sounds for each of these 3 events, but you may choose to play any sound on your system. To do this, press the button for the event you wish to change the sound for, and select a sound from your computer.

Note: Only PCM wave files are supported. This is the standard wave file format. This includes most wave files, but some wave files are not encoded in PCM format, but in another format. If this is the case, you will receive a message about the file not being a valid format.

Changing Display options

In Preferences dialog choose the General Tab

Here you can set general options that change how a location is displayed on screen, such as 12 or 24 hour format, show seconds etc.
