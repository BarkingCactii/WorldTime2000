Skins

 

·      Menu Action->Load Skin

·      Menu Action->Remove Skin

·      Toolbar

 

WorldTime2000 supports application skinning where you can change the overall look of the program by loading different skins. WorldTime2000 is compatible with standard ActiveSkin skins

To change the skin, choose Load Skin and pick a new skin. WorldTime2000 will adopt the look of the new skin immediately.

If you prefer to have the standard Windows look, choose Remove skin. Note: You will have to restart WorldTime2000 if you choose this option.
