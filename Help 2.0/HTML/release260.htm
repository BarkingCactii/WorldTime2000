Release  2.60

·      This version finally uses HTML help and removes the kludgy HTML window controls of previous versions. This provides a standardized interface, and allows searches to be made.

·      Many code changes have taken place to make WorldTime2000 more reliable. This includes optimizations and changes to the registration to help make the program more robust and compatible. As a result, the installer is roughly 50K smaller than the previous version
