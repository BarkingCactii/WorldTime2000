Release 2.10

·      Tool tips for toolbar buttons

·      Auto selection has been turned off. Locations can now only be selected by left clicking on the location

·      Tool tips don't display if mouse is not in the application area

·      Locations can be sorted by name, country or time from GMT in ascending or descending order

·      Save option added to menu, so you elect when to save data, rather than previously saving only when the program exited.

·      The offset hours column can now be hidden to reduce the display
