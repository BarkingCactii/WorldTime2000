Status Icon Meaning

 

For each location, there can be up to 3 status icons displayed to the left of the locations time. The status icons signify

1.  Sun and night phase This gives a quick visual confirmation for the time of day or night. There are 14 different stages from dawn to sunset, twilight to dawn.

2.  Alarm Status. This icon is displayed if there are any alarms applicable for this location

3.  Daylight Savings. This icon is displayed if this location is currently in daylight savings time
