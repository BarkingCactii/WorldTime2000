Moving locations

 

·      Hotkey None

·      Menu None

·      Context Menu Move

 

Right click on a location to move it. This allows you to move locations within the display. In essence, its a tool to let you order your locations on the display. When you add a location to the display, it adds the location to the bottom of the list. You may want to move that location to another position. To do so, choose Move. The location you wish to move will start flashing with a message 'Select new location'. Using the mouse, select a new slot where you would like the location to be moved to
