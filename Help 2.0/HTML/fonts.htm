Fonts

 

·      Hotkey F

·      Menu Action->Fonts…

·      Contect Menu Set Font for this location

 

There are 2 types of Font settings. You can set the Font for all locations or set the font for an individual location. To set the font for an individual location choose Set Font for this location by right clicking on the location you wish to change.

Once you have chosen a new font, WorldTime2000 will then adjust the display to accommodate the size and height of the new font. You may have to resize the window to achieve the best affect.
