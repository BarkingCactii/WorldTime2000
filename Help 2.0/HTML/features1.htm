Features

·      View hour differences from any location

·      Click on a location, and hourly differences will be calculated from this new reference. 

·      Set alarm reminders for any location, and optionally run an application. 

·      A What if feature is included to make it easy to schedule phone calls or meetings for anytime, not just the current time. 

·      Comes with time details for over 400 locations, including cities and time zones with the ability to modify these and add more. 

·      Display will automatically adjust itself if you change the window size. 

·      Your choice of fonts, colors, wallpapers and sounds can give this program a very individual, cool look.
