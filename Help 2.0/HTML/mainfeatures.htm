Main Features

 

·      Comes with time details for over 1200 locations, with the ability to modify these and add more locations. 

·      Locations include rules for Daylight Savings

·      Display will automatically re-proportion itself if you change the window size. 

·      Your choice of fonts, colors, wallpapers and sounds can give this program a very individual, cool look.

·      Each location can be individually customized

·      View hour differences from any location – just select the location as your reference point

·      Set unlimited number of recurring alarms/reminders for any location, and optionally run an application or display a pop up window with your own message. 

·      A What if feature is included to make it easy to schedule phone calls or meetings for anytime, not just the current time. 

·      Supports application skinning

·      Tooltips for each location show extended information

·      Minimize to the System Tray

·      Time Sync facility
