The Spinning Sphere

In Preferences dialog

This refers to the rotating earth you see spinning in the background. You either gotta love it or hate it.

There are a number of options here

1.   Display on screen - turns the sphere on or off

2.   Moving ? - if you choose to display on screen, then you can view either a spinning globe that sits in the corner, or one that moves across the screen into the distance.

3.   Number of Frames - If you choose a moving sphere, this is the number of frames of animation before it disappears into the distance. Larger numbers mean a smoother look, at the expense of taking more processing time.

When WorldTime2000 is set up exactly how you want it, and you no longer need to view the main display any more, you can close or minimize the application window. At this time, WorldTime2000 will continue to run in the system tray. The system tray is the area on the bottom right of your screen containing a list of icons from different applications.

There are several things you can do with this icon.

1.   Double left clicking on this icon will open the main WorldTime2000 window

2.   Right clicking on this icon will present a mini-menu where you can do several things




This provides a quick way to view or set important information without opening the main window.

WorldTime2000 uses the same icon picture in the system tray as its shortcut on the desktop. If you keep the mouse over the WorldTime2000 for a few seconds, a tooltip will popup describing:

1.   How long WorldTime2000 has been running

2.   How many alarms are pending ( 1 alarm per location )

3.   Whether DST for your home is on or off
