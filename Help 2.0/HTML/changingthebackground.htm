Changing the Background

In Preferences dialog, choose the Display Tab

You can set WorldTime2000 background to be either a simple color or a wallpaper

With wallpapers, the bitmap must be a bitmap ( BMP ) file. If the bitmap file is smaller than the display, the background is drawn using a tiling method, thus making sure the background is always textured, even for small wallpapers. Alternatively, you may choose to display pictures larger than the display, to make a scenic background look. The choice is entirely up to you.
